Other inputs - not Entity related.
Freeze - Stop movements
Unfreeze - Enable movements again
Breakable - Make it breakable/destroyable by force
Invincible - Indestructible/Godmode
GravityUp - More gravity means more force to the ground
GravityDown - Less gravity means less force to the ground
SizeUp - Increase the size of the entity
SizeDown - Decrease the size of the entity
SpeedUp - Increase the movement speed of walkable entities
SpeedDown - Decrease the movement speed of walkable entities
Solid - Make the entity passable. You can walk through it
UnSolid - Upperside of Solid
Touch - Similar to +use (press e)
Ignite - Burn it
Visible - Make visible
Invisible - Make invisible
Activate - Activate entity
ChangeSkin - Change your own skin to the entities skin
Hurt - Do 100000 damage
HealthToOne - There is no need for a description
HealthToFully - There is no need for a description
Remove - There is no need for a description
